WebAudioXML

Click and slide to play

  • Pointer relative X position controls pitch
  • Pointer relative Y position controls cutoff frequency of biquad filter

WebAudioXML is a javascript library that makes it possible to create Web Audio applications using XML syntax. It is a PhD research project by Hans Lindetorp. Please follow me on https://hans.arapoviclindetorp.se and https://www.facebook.com/hanslindetorpresearch/
